[MANUAL DE INSTALAÇÂO E USO]

1. Baixe o projeto no seu computador.
2. Extraia o conteúdo da pasta "Programa" para uma pasta do seu computador.
3. Crie um atalho do executável "Catalogo.exe" na área de trabalho.
4. Execute o programa através do atalho.
5. Ao iniciar o programa você receberá um aviso de erro de conexão com o MYQSL.
6. Confirme o diálogo, e o programa vai abrir.
7. Clique sobre o botão "Configurar".
8. Informe as credenciais de acesso ao banco de dados OLFA.
9. Selecione a pasta das imagens.
10. Selecione a pasta de saída do PDF gerado.
11. Selecione a tabela de preço.
12. Selecione as definições de Font.
13. Clique sobre o botão "Gerar PDF".
14. Se tudo ocorrer bem será gerado o arquivo PDF na pasta que você selecionou.

OBSERVAÇÕES

O nome dos arquivos de imagem deverão ser o código do produto
Exemplo: para o produto "840010U" renomeie o arquivo para "840010U.jpg"...
